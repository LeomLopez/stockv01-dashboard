Sobre o Sistema Gestão Stockv01

O que é o Stockv01?

O Gestão Stockv01 é uma aplicação web moderna para gerenciar o estoque de produtos. Permite criar, atualizar e excluir produtos com facilidade.

Desenvolvedor

Desenvolvido por Leomar López.

Tecnologias Utilizadas
  • Backend: Flask (Python)
  • Banco de Dados: PostgreSQL
  • Frontend: Bootstrap 5
  • ORM: SQLAlchemy
Principais Funcionalidades
  • ✓ Gestão completa de produtos
  • ✓ Interface responsiva com Bootstrap
  • ✓ Banco de dados PostgreSQL robusto
  • ✓ Fácil de expandir e personalizar
