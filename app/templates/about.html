Acerca de Stockv01 Management

¿Qué es Inventory Web?

Stockv01 Management es una aplicación web moderna para gestionar el inventario de productos. Permite crear, actualizar y eliminar productos con facilidad.

Tecnologías Utilizadas
  • Backend: Flask (Python)
  • Base de Datos: PostgreSQL
  • Frontend: Bootstrap 5
  • ORM: SQLAlchemy
Características Principales
  • ✓ Gestión completa de productos
  • ✓ Interfaz responsive con Bootstrap
  • ✓ Base de datos PostgreSQL robusta
  • ✓ Fácil de expandir y personalizar
